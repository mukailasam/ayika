# Ayíká: Simple Environment Loader for Go

Ayíká means _Environment_ in Yoruba.

It is a tiny zero-dependency `.env` loader for Go projects.  
It automatically loads `.env` when imported and supports logging.

## Features

- Automatic `.env` loading (no function call needed)
- Ignores comments and malformed lines
- Optional debug logging
- Zero external libraries

## Installation

```
go get github.com/mukailasam/ayika
```
